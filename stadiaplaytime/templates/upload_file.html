Welcome to the calculator of Stadia play time!

  1. Download your Stadia data from Google Takeout (a tip is to only select Stadia, so that you get it faster).
  2. In the zip file, go to Takeout\Stadia\GAMING\GAME_DATA and unzip the GAMER_HISTORY.json file.
  3. Then upload your GAMER_HISTORY.json file using below form.
This page is not associated with Google in any way.
Please note that your uploaded data can be saved for some time on the server.
